# Инструкция по настройке Qwen Wallet

## Готово к использованию

Приложение готово к работе с GitHub синхронизацией.

## Настройка GitHub

1. **Создайте репозиторий на GitHub** (например: `qwen-wallet-cards`)

2. **Добавьте файл `cards.json`** из этого репозитория в корень вашего репозитория:
   - Файл уже создан локально: `D:\DISTR\AI_Project\Qwen_Wallet\cards.json`
   - Загрузите его на GitHub

3. **Настройте URL в `index.html`** (строка ~1391):
   ```javascript
   const GITHUB_RAW_URL = "https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/cards.json";
   ```
   Замените `YOUR_USERNAME` и `YOUR_REPO` на ваши данные.

4. **Разместите приложение на GitHub Pages** (опционально):
   - Настройте GitHub Pages для вашего репозитория
   - Приложение будет доступно по URL и будет обновляться автоматически

## Что сделано

- Упрощен главный экран - остались только поиск, часто используемые и все карты
- Удалены страницы Каталог, Выгода, Ещё и нижняя навигация
- Отключено добавление карт в приложении - данные только из GitHub
- Реализована синхронизация с GitHub при запуске и каждые 5 минут
- Штрихкоды и QR-коды работают корректно
- Создан `cards.json` с 8 картами российских магазинов

## Тестирование

1. Откройте `index.html` в браузере
2. Должны отобразиться карты из `cards.json` (если не настроен GitHub, покажутся тестовые карты)
3. Нажмите на карту - откроется детальный просмотр с штрихкодом/QR
4. Для синхронизации с GitHub настройте URL в коде

## Примечания

- Карты хранятся в localStorage для оффлайн-доступа
- При изменении `cards.json` на GitHub приложение обновит данные автоматически через 5 минут
- Для публикации на GitHub Pages используйте ветку `main` или `gh-pages`
